🎬 Before You Record — Production Checklist

💻
Screen Setup

Set resolution to 1920×1080. Use Chrome in full-screen. Hide bookmarks bar. Close all other tabs. Use a clean browser profile with no extensions showing.

🎙️
Audio

Record in a quiet room. Use a USB condenser mic or headset. Record narration separately and mix in post for cleaner audio, or use live narration if comfortable.

🖱️
Mouse & Cursor

Use a spotlight/cursor-highlight tool (e.g., Mouseposé on Mac, Cursor Spotlight on Windows). Move the mouse slowly and deliberately between elements.

🗃️
Demo Data

Use demo credentials: demo@clarity.health / Demo2026!. Patient: Sarah Johnson, DOB 03/14/1985. All data is pre-seeded and reset-safe.

⏸️
Pacing

Pause 1–2 seconds after navigating to a new screen before speaking. This gives viewers time to orient. Don't rush — aim for a calm, confident delivery.

🎨
Post-Production

Add a lower-third title bar for each section. Use smooth fade transitions (0.3s). Add background music at ~20% volume (royalty-free, upbeat corporate).
