## Contingencies

### If Timeline Changes

If the program timeline or milestones shift >2 weeks, schedule a brief (15-min) re-approval call with all 4 executives to confirm continued commitment.

### If Budget Increases >10%

If costs exceed $210K, CEO + CFO must approve incremental budget with written rationale.

### If Scope Changes

If scope expands (e.g., add PI or P criteria, include mobile apps), all 4 executives must re-sign charter amendment.

## Appendix: What You're Signing

By signing this charter, you're committing to:

✅ **Budget:** Authorize $147–210K spending on audit + personnel  
✅ **Timeline:** Commit to 18-month program (Q3 2026–Q4 2027)  
✅ **Resources:** Release staff for 0.95 FTE-year of compliance work  
✅ **Support:** Participate in evidence collection, policy reviews, auditor interviews  
✅ **No scope creep:** Don't expand audit scope without re-approval  
✅ **Monthly reporting:** Receive monthly compliance dashboard  
✅ **Auditor access:** Grant auditor access to systems, logs, interviews  

**What you're NOT signing:**
- ❌ Guarantee of zero findings (audit findings are normal)
- ❌ Guarantee the report will be issued (only happens if controls operating effectively)
- ❌ Unlimited budget (increases >10% require re-approval)
- ❌ Personal liability (institutional commitment only)
